# 🎯 飞书协同版智能招聘漏斗 (Feishu Intelligent Recruitment Funnel)

这是一个用于 OpenClaw 多智能体生态的企业级开源技能 (Open Skill)。

[🇨🇳 中文版文档](./README.md) | [🇺🇸 English Documentation](./README_EN.md)

## 🌟 核心特性 (V3.2 重磅升级)
- **红蓝对抗架构 (Good Cop / Bad Cop)**: 师爷猫负责极度挑剔的“X光打假”与审讯陷阱布置，大主管负责务实的兜底与降维匹配。
- **五大反诈红线体系**: 彻底查杀 AI 生成模板、伪 Owner（把参与包装成主导）、时间压缩造假（3个月完成N个大项目）以及幽灵项目。
- **面试陷阱生成器 (Interview Traps)**: 专门为面试官生成致命的连击拷问，带着问题去面试，一秒刺穿候选人的包装。
- **原生多模态解析**: 彻底废除 OCR，直接调用系统大模型能力看透 PDF 简历排版与隐含逻辑。
- **飞书多维表格 (Bitable) 状态机**: 自动建表（Auto-Provisioning），全量支持 JD 库（多岗位并发）与动态时间线防伪。
