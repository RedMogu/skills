# 🎯 Lark (Feishu) Intelligent Recruitment Funnel

An enterprise-grade Open Skill for the OpenClaw multi-agent ecosystem.

[🇨🇳 中文版文档](./README.md) | [🇺🇸 English Documentation](./README_EN.md)

## 🌟 Core Features
- **Red/Blue Team Architecture (Good Cop / Bad Cop)**: The Sub-Agent (Advisor Cat) acts as a highly critical "X-ray" auditor, while the Main Agent acts as a pragmatic manager for fallback matching and role downgrading.
- **Full-Stack Role Coverage**: Guards against tech buzzword stuffing (e.g., "RAG", "High Concurrency") and non-tech grand narratives (e.g., "0-to-1 ecosystem").
- **Native Multi-modal Parsing**: Completely replaces OCR by utilizing native LLM vision capabilities to understand PDF resume layouts and implicit logic.
- **Lark Bitable State Machine**: Auto-provisioning tables, full JD library support (concurrent multi-role matching), and dynamic anti-fraud timeline verification.
